# Prompt Sharing

一个高质量提示词资源库，帮助你更好地与 AI 协作。

## 项目介绍

本项目收集和分享经过实践验证的优质提示词模板，涵盖设计、优化等常见场景。所有提示词均遵循最佳实践，可直接使用或根据需求调整。

## 提示词模板

### 1. 提示词设计顾问
**文件**: `prompt_design_consultant.md`

从零开始设计全新的 AI 提示词，提供需求挖掘、方案设计、互动完善等专业服务。支持多种风格方案选择，总能找到最适合你的设计。

### 2. 提示词优化助手
**文件**: `prompt_optimization_assistant.md`

专业的提示词分析和改进工具，提供快速诊断、完整分析、深度优化三种服务模式。从清晰度、针对性、完整性、可操作性、抗干扰性五个维度评估提示词质量，并提供具体优化建议。

### 3. 项目成长伙伴
**文件**: `project_partner_jtbd.md`

基于 JTBD（Jobs to be Done）框架的项目助手，帮助 1-3 年经验的开发者从想法到落地。擅长需求分析、项目规划、开发指导和经验总结，引导你聚焦核心价值，避免功能堆砌。

### 4. 智能写作助手"小7"
**文件**: `xiao7_intelligent_writing_assistant.md`

专注于 AI 提效和创意内容创作的写作伙伴。擅长 AI 知识科普、新闻资讯、Prompt 技巧教学、工具推荐、效率提升和避坑分享。风格轻松有趣，像朋友一样交流。

## 使用方法

1. **设计新提示词**：使用 `prompt_design_consultant.md` 中的提示词与 AI 对话，获得量身定制的提示词方案。

2. **优化现有提示词**：使用 `prompt_optimization_assistant.md` 中的提示词，分析和改进你的提示词，提升 AI 输出质量。

3. **复制使用**：所有提示词均为完整可用版本，可直接复制到 AI 工具中使用。
